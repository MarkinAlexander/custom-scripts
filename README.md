# custom-scripts
Различные sh скрипты


### Починка скачки и открытия с гитхаба

```bash
curl --resolve raw.githubusercontent.com:443:185.199.108.133 -O https://raw.githubusercontent.com/MarkinAlexander/custom-scripts/main/gh_fix.sh && bash gh_fix.sh
```

Явно указывает с какого айпи адреса качать скрипт. А после запускает через bash. Если баш отсутствует в системе, то ставить через свой пакетный менеджер.

Скрипт проверялся на чистом Linux и Keenetic Entware. В результате работы он выдаст все рабочие айпишники и предложит выбор если адресов больше одного. Если только один, то автоматом его пропишет.
